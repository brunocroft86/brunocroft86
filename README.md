# Olá, eu sou o Bruno Galvão 👋

**Engenheiro de Software | Desenvolvedor Full-Stack | Professor de Informática**

Apaixonado por tecnologia, educação e por mudar a vida das pessoas através da programação. Formado em Engenharia de Software e residindo no Rio de Janeiro, divido meu tempo entre construir soluções robustas para a web e lecionar, compartilhando conhecimento em projetos sociais como o "Mulheres Conectadas" e na formação de novos profissionais.
